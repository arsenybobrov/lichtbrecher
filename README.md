# Lichtbrecher

### installation
```yarn install```

Copy ```api/prismic/configuration-dummy.js```, rename it to ```api/prismic/configuration.js``` and edit the vars.


### development
```yarn dev```


### deployment
```yarn build```


### Prismic
Custom types must be:

 - homepage
 - page
 - global_config
 
 
 ### usefull third party
 The following libs are not neccessary for the setup to work but are very nice:
 
 - gsap
 - react-scroll
 - lodash
